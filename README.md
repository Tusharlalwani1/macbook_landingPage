# 🍎 MacBook Landing Page
 
A visually rich, Apple-inspired landing page built with React, Three.js, and GSAP, featuring a 3D product viewer, scroll-driven animations, and a cinematic showcase section.
 
🔗 **Live Demo:** [macbook-landing-page-pearl.vercel.app](https://macbook-landing-page-pearl.vercel.app)
 
---
 
##  Features
 
- **3D Product Viewer** : Interactive MacBook model built with Three.js and React Three Fiber. Switch colors and sizes in real time.
- **Scroll Animations** : GSAP ScrollTrigger powers smooth, cinematic transitions throughout the page.
- **Video Showcase** : Full-screen video section with an SVG mask reveal effect.
- **Performance Section** : Animated graphics performance highlights with parallax imagery.
- **Highlights Masonry** : Apple-gradient bordered cards with responsive grid layout.
- **Fully Responsive** : Optimized for desktop, tablet, and mobile with Tailwind breakpoints.

## 🚀 Getting Started
 
### Prerequisites
- Node.js 18+
- npm or yarn
### Installation
 
```bash
# Clone the repo
git clone https://github.com/Tusharlalwani1/macbook_landingPage.git
 
# Navigate to project
cd macbook_landingPage
 
# Install dependencies
npm install
 
# Start dev server
npm run dev
```
 
Open [http://localhost:5173](http://localhost:5173) in your browser.
 
### Build for Production
 
```bash
npm run build
npm run preview
```
